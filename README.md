#Registration Form

##Dependencies

###Through cdns:
####Angular 1.4.9
####Angular ui router 0.2.17

Must be installed:
####Python ^2.7.10

#### This was created using Angular, due to Chrome's setup the files need a server to prevent cross origin request error alerts. To fix this go to the app/client directory and run:

```
live-server
```

This will set up a simple http server through python and stop the cross origin request alerts.
